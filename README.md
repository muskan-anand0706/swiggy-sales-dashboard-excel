# Swiggy Sales Dashboard | Excel

**Brief One Line Summary:**  
Interactive Excel dashboard to analyze sales performance, order trends, customer ratings, and average order value.

**Overview:**  
Analyzed over 197,400 sales and customer records to uncover performance trends and customer engagement patterns. Built an interactive Excel dashboard for multi-dimensional exploration of sales data.

**Problem Statement:**  
Understanding sales trends, order behavior, and customer ratings is crucial for data-driven business decisions. This project focuses on identifying key patterns in Swiggy sales data to support performance analysis and decision-making.

**Dataset:**  
Publicly available sales and customer dataset (raw and cleaned Excel files included).

**Tools and Technologies:**  
Excel | Pivot Tables | Charts | Slicers

**Methods:**  
- Data cleaning and preparation in Excel  
- Creation of pivot tables and charts for key metrics  
- Interactive dashboard with slicers and filters for multi-dimensional exploration

**Key Insights:**  
- Analyzed overall sales performance using total sales and order volume  
- Evaluated customer engagement through average ratings, rating count, and average order value  
- Insights support data-driven business and operational decision-making

**Dashboard / Output:**

- 3 slicers/filters for easy exploration  
- Visualizations of overall sales performance and customer engagement metrics  

**How to Run this Project:**
1. Open the Excel file (.xlsx)
2. **Click "Enable Editing"** at the top to activate interactive features
3. Use the dashboard slicers/filters to explore sales trends and customer behavior
4. Review visual insights and pivot table summaries

**Results & Conclusion:**  
- Clear view of overall sales performance and customer engagement patterns  
- Dashboard summarizes total sales, orders, average rating, rating count, and average order value  
- Supports data-driven business and operational decision-making
